# Redis

Redis 是由 [[antirez]] 建立的 in-memory data structure store，常用於 cache、queue、session store 與即時資料結構。本 wiki 目前不是以 Redis 本身作為資料庫主題，而是因為 antirez 在 2026-04-26 的 [[DeepSeek]] V4 llama.cpp 實驗中被提及。

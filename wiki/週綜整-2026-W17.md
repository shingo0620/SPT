⚠️ **真相**（2026-05-01 發現）：skills.sh 自 **2026-04-19** 起停發新 pick——這些「連 N 日精選」全部為 fetcher 靜默 fallback 假象。**安裝數縱向變化是真實的**（從 ast-grep 頁面即時擷取），但「skills.sh 連續精選」「擴散 S 曲線」「anchor skill 候選」「+186 與 DeepSeek V4 因果論述」均屬基於誤判的虛構推論，已撤回。詳見 [[src-skills-picks-2026-04]] 校正聲明。

### 04-25 lint 誤判 raw 為重複檔

「我似乎已经看见了，游戏引擎的未来！.md」原為 16 行 Obsidian Clipper YouTube metadata 有效來源，被誤刪。**04-26 修正並 ingest 為 [[src-遊戲引擎的未來]]**。

**教訓 → memory feedback**：「lint 判斷重複檔前必須 wc + diff」「URL 標籤檔（1 行）vs 完整 metadata 必須區分」。

### GitHub Trending 連日詐騙 repo 衝榜

W17 期間 GitHub Trending 持續出現 ❌ 加密詐騙 repo（Flash-USDT-Sender、Exodus-Fake-Balance 等），**連七日 spam 模式**（04-19 起到 04-25），形成系統性現象。HN 04-21 #4「GitHub 假星經濟」（422 分）為跨來源相互印證。

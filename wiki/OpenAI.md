# OpenAI

> 本頁統整本 wiki 多處提及的 OpenAI 脈絡；來源為空表示為跨來源歸納。

## 一句話定位

2015 年成立的 AI 研究組織（後轉商業化），**以 GPT 模型線、ChatGPT 消費級產品、Codex 開發者工具線**主導 LLM 商業化浪潮——是 [[Anthropic]]、[[Google]]、[[DeepSeek]] 的主要競爭者。

## 核心策略

- **閉源 + API 商業化**——與 [[DeepSeek]] / [[Qwen]] 開源策略對立
- **Codex 全流程 autopilot**——「不只生 code，還能跑、測、debug」進化路線
- **多模態先行**——ChatGPT Images 2.0 等將 LLM 與生成式擴展整合
- **Microsoft 深度綁定**——Azure 雲端基礎設施、Microsoft 365 Copilot 整合
